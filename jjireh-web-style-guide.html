Paragraph Text
This site uses the Google font Oswald in 300, 400 and 600 weights. the standard text size is 1.25em and color is #63666a. Bold and header text is slightly darker — using #4b4d50.

Small text size is 1.1em. It can be used as secondary or sidebar body copy, detailed information or links.

X-small text size is 0.875em and should be used sparingly.

Other company logos

Use of other company logos on the website or in emails requires adjustments to avoid conflicting with the main brand.

A CSS class of ".partner" will automatically apply grayscale and opacity effects. To replicate this in Photoshop add a hue/saturation adjustment layer – reduce the saturation to -100 and increase the lightness to +35.
